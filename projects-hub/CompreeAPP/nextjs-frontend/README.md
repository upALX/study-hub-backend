## How to use 🫁

**1 - Clone this repo**
```
git@github.com:upALX/Store-API.git
```
**2 - Before run the Compree, you need serve the images on a server to emulate an CDN server:**
```
npx serve -l 9000 images
```

**3 - With all done, run the application (certify that images are UP)
```
npm run start
```
